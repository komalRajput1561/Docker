
# Learning Node.js

## Introduction:
     Node.js is a cross-platform environment and library for running JavaScript applications which is used to create networking and server-side applications.


## Features of Node.js

    1. Single-threaded

    2. I/O is Asynchronous and Event Driven

    3. Extremely fast

    4. Highly Scalable

    5. No buffering
